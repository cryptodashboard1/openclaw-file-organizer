# OpenClaw Command Skill

## What this folder is
Skill package that lets OpenClaw run Auto-Organizer through natural commands.

## What's inside
- `SKILL.md`: skill instructions and command flow

## When to edit this folder
- You are changing user-facing command behavior or orchestration wording.

## What not to put here
- Tool registration code (`openclaw-plugin-auto-organizer/`)
- Runtime API handlers (`apps/`)

## Most important files
- `SKILL.md`
